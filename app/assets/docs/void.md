# The Void

There are no more docs. Everything else is application-specific. Routing,
components, deployment, OOB swaps — that's the framework surface area.
The rest is just Go code.

Write handlers. Query databases. Emit HTML. There's no plugin system, no
middleware ecosystem, no "app router vs pages router" schism to navigate.
You've reached the end of the opinionated part. What comes next is yours.

Go wild.
